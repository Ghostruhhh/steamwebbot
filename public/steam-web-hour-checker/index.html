Rust · Battlemetrics

To show tracked time on BM-indexed Rust servers, put a token in your .env as BATTLEMETRICS_TOKEN (get one free) and restart the server. This is independent of your Steam Web API key.

Battlemetrics profile ID (optional)

If Steam never auto-links, open the player on battlemetrics.com and copy the number from the URL (/players/123456). Saving it here sends bm_player= on every lookup — best when you’re viewing your own profile; for someone else, use their BM id or expect no match.
