PC serial checker

            

              Click Scan this PC to pull SMBIOS/WMI identifiers from the Windows machine running server.js — same layers most hardware tools read.
              Open the hub via localhost or 127.0.0.1; this API is blocked for other computers on your LAN.
            

            

              Important: Nothing here “activates” Windows or games. VMs and some OEM BIOSes show generic serials (“To be filled by O.E.M.”). Run with Node on bare metal when you care about motherboard/disk dumps.

Startup apps

            

              Programs registered to start with Windows on this PC, via WMI
              Win32_StartupCommand (registry Run keys, RunOnce where listed, and Startup folder shortcuts).
              The list loads automatically when you open this tab; the API only responds when the hub is opened on localhost or 127.0.0.1 (same as the footer URL).
              Use Download report file to save JSON + embedded log for the Home paste viewer.
            

            

              Read-only. Disabling entries is done in Task Manager → Startup apps, the legacy shell:startup folder, or Registry — not from this hub.
            

            

              
              
              
            

            

              
            

            

            

          

        

        

          

            
RAT, malware & cheat hints

Products & keys

            

              Create products and mint GP-style keys. Everything lives in
              data/license-store.json.
              Keep this dashboard on this PC only while the hub is running (localhost).
              Buyers: give them /loader/ — they paste the key once, tie it to that computer, then grab the installer if you’ve wired one up.
            

            

              Heads‑up: keys aren’t Stripe or encryption — treat data/
              like a spreadsheet you must back up yourself.
            

            

              One computer per key after the first activation. Buyers can’t casually move the license to endless PCs unless you reset that key row.
              Time‑limited keys only stop working inside your installer if your installer periodically checks back with your hub — the file sitting in Downloads alone can’t magically switch off.
